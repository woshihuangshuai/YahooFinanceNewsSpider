# YahooFinanceNewsSpider
基于scrapy编写的python网络爬虫，用于爬取Yahoo Finance上的财经新闻。
爬取的新闻包括以下三类：

 * 以sector分类的新闻

 * 以industry分类的新闻

 * S&P500中包含的所有公司，按公司分类的新闻数据。

# Environment

 * Python 2.7
  
 * MongoDB
  
 * FireFox
  
 * pip install scrapy / apt install python-scrapy
  
 * pip install -U selenium
  
 * pip install pymongo
